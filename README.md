# Mesto

Интерактивная веб-страница для работы с карточками, позволяющая динамически добавлять, удалять карточки и отмечать
понравившиеся.

### Основные возможности

- Редактирование информации о пользователе.
- Создание карточек пользователями.
- Удаление доступно только автору карточки.
- Установка и снятие отметок «нравится».
- Валидация форм с использованием JavaScript.
- Модальные окна для просмотра изображений в большом формате и работы с формами.
- Удобный и интуитивно понятный интерфейс.

### Используемые технологии

- **HTML5/CSS3**
- **JavaScript (ES6)**
- **Методология БЭМ**
